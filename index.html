Elementos base del documento

Resumen de elementos obligatorios
Elemento Descripción
<!DOCTYPE html> Declaración del documento HTML5
<html lang="es"> Idioma del documento
<head> Metadatos, título, enlaces CSS
<body> Contenido visible
<header> Encabezado principal
<section> Bloque temático
<footer> Pie de página
<table> Tabla con al menos 3 filas

Etiquetas nuevas exploradas: <figure> y <aside>

Nota al margen (uso de <aside>)

<aside> contiene información complementaria al contenido principal. Aquí, por ejemplo, dejamos el enlace a la página de la Universidad Politécnica Salesiana y al repositorio de la página.
